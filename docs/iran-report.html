米・イスラエル対イラン 軍事衝突

情勢報告書 — Situation Report
2026年3月5日（木）第6日目

← 株式市場レポートに戻る
速報（3月5日）: テヘランで新たな爆発が報告。米上院はトランプ大統領のイラン軍事行動に対する議会承認を求める決議案を47対53で否決。ヘグセス国防長官は「まだ始まったばかり」と発言し、作戦の加速を宣言。イランの対艦ミサイル・ドローン発射量は開戦初日比で86%・73%減少。

紛争の概要

作戦名と目的

  • イスラエル側: 「Operation Roaring Lion（獅子の雄たけび）」
  • 米国側: 「Operation Epic Fury（壮絶な怒り）」
  • イラン報復: 「Operation True Promise IV（真の約束IV）」
  • 目的: イランの核開発阻止、体制転換（レジームチェンジ）
  • 開始: 2026年2月28日（土）、テヘラン・イスファハン・コム・カラジ・ケルマンシャーへの同時攻撃

主要な被害

  • ハメネイ師暗殺: 最高指導者アリー・ハーメネイーの司令部が破壊され死亡。3日間の国葬がマシュハドで予定
  • イラン軍事力: 米軍はイラン海軍艦船20隻以上を撃破（最大の潜水艦含む）。イランの海軍・空軍・防空レーダーは事実上壊滅とトランプ大統領が主張
  • 核施設: ナタンツ核施設に被弾、イスファハンの核施設2棟にも損傷（IAEA確認）。放射性物質の漏洩は未確認
  • 女子校攻撃: ミーナーブの女子校へのイスラエル軍攻撃で180人死亡との報道
  • IRIS Dena撃沈: 3月4日、スリランカ沖でイラン海軍フリゲート艦がUS潜水艦により撃沈。フォークランド紛争以来初の潜水艦による実戦撃沈
  • IAF空爆: イスラエル空軍はイラン31州中24州で1,600回以上の出撃、4,000発以上の弾薬を投下

時系列

2月28日（土）— 第1日

米・イスラエルがテヘラン・イスファハン等へ同時攻撃開始。ハメネイ師の司令部を破壊、最高指導者を暗殺。イランは即座に報復を宣言

3月1日（日）— 第2日

イランが「真の約束IV作戦」を発動。500発以上の弾道ミサイルと2,000機以上のドローンでイスラエル・米軍基地を攻撃。テルアビブで40棟以上損壊。ベイト・シェメシュのシナゴーグに着弾し9人死亡。ホルムズ海峡の封鎖を宣言

3月2日（月）— 第3日

ヒズボラがイスラエルにミサイル・ドローン攻撃を開始。イスラエルは「ヒズボラによる正式な宣戦布告」と断定し、レバノンへの空爆を開始。イラン側死者555人に

3月3日（月）— 第4日

イスラエルがイラン国営放送IRIB本部・国会議事堂を空爆。専門家会議（次期最高指導者選出機関）の建物も攻撃。米大使館3館（サウジ・クウェート・レバノン）が閉鎖。米軍死者6人に

3月4日（火）— 第5日

スリランカ沖でIRIS Dena撃沈（死者80人以上）。レバノンでの死者50人超。イスラエルが「新指導者も標的」と警告。NATO防空システムがトルコ方面のイランミサイルを初迎撃

3月5日（水）— 第6日

テヘランで新たな爆発。米上院が戦争権限決議を否決（47-53）。ヘグセス国防長官「まだ始まったばかり」。イラン内陸部への攻撃拡大を予告。レバノンでは子供7人を含む新たな死者

イランの報復攻撃

攻撃対象と範囲

  • イスラエル: テルアビブ・エルサレムに弾道ミサイル。テルアビブで40棟以上損壊
  • 米軍基地: ヨルダン・クウェート・バーレーン・カタール・イラク・サウジ・UAE所在の27基地を攻撃。バーレーンの第5艦隊司令部に複数回着弾
  • 湾岸諸国: ドバイのブルジュ・アル・アラブに被弾。サウジアラムコ石油施設にミサイル攻撃。バーレーン国際空港にドローン攻撃
  • トルコ: インジルリク米空軍基地へのミサイル攻撃（NATOが迎撃）
  • IRGC: 27の米軍展開基地に攻撃実施と発表。「攻撃が続けば全ての経済拠点を標的にする」と警告

各国・国際機関の反応

国連

  • 安保理が2月28日に緊急会合を開催
  • グテレス事務総長が「国際平和への重大な脅威」と危機感を表明、即時交渉を要求
  • イラン国連大使が「100人以上の子どもが死亡、人道に対する罪だ」と訴え
  • IAEA（国際原子力機関）はイランが核兵器に「数日〜数週間」の距離にあったとの米・イスラエルの主張を否定

中国

  • 「軍事行動の即時停止」と「対話・交渉の再開」を呼びかけ
  • ただし、イランへの積極的な軍事支援は示さず、抑制的な対応
  • 中国は世界最大の石油・ガス輸入国であり、原油の1/3をホルムズ海峡経由で調達。経済的影響は深刻
  • 西側制裁と湾岸諸国との関係に配慮し、正面からの米国批判を回避

ロシア

  • 安保理で米・イスラエルの攻撃を「国際法違反」と非難
  • ブシェール原発が攻撃の脅威にさらされていると警告
  • ウクライナ戦争でイランを準同盟国としてきたが、軍事支援の余力なし
  • 「同盟国を守れない大国」というメッセージを発信するジレンマに直面

その他の国

  • 日本: 高市早苗首相がX上で「あらゆるリスクに備え万全の対応」を表明。NSC臨時会合を開催
  • カナダ: カーニー首相が軍事的関与の可能性を「完全には排除できない」と発言
  • レバノン: 政府がヒズボラの軍事活動を公式に禁止し、武器の引き渡しを要求
  • カタール: IRGC関連スパイ10人を逮捕。ドーハ上空でミサイルを迎撃
  • 米国内: ルビオ国務長官が中東在住の米国市民に即時退去を勧告

経済・エネルギーへの影響

ホルムズ海峡危機

  • イランは海峡封鎖を宣言。実際にはドローン攻撃で保険会社が戦争リスク保険を撤回し、事実上の通航停止に
  • タンカー交通量が約70%減少後、ほぼゼロに。150隻以上が海峡外で停泊
  • 世界の石油供給の約20%が影響。RBCの専門家は「1970年代の石油禁輸以来最大のエネルギー危機」と評価
  • トランプ大統領がタンカーへの保険提供と海軍護衛を表明。ただし専門家は実効性に懐疑的

市場への影響

  • 原油: Brent原油が$83/バレルまで上昇（開戦前$73から10%超上昇）。$100突破の可能性も
  • 天然ガス: 欧州先物が30%急騰。カタールのLNG施設が攻撃を受け生産停止
  • 株式: アジア市場が大幅下落。韓国KOSPIは12%超の暴落で取引停止。ソウル市場で記録的な売り
  • 米ガソリン: 全米平均ガソリン価格が1日で12セント/ガロン上昇（4年間で最大の値上がり幅）
  • 航空: 100万人以上が移動に影響。中東発着便1,900便以上が欠航。ドバイ空港で大混乱
  • イラク: ホルムズ海峡封鎖で石油輸出不能となり、主要油田の生産停止を余儀なくされる

米国内政治の動き

  • トランプ大統領は戦争が「4〜5週間」で終わるとの見通しを示す
  • 攻撃の正当化理由が「核開発阻止」「体制転換」「差し迫った脅威の排除」「イスラエルへの追従」と複数あり、一貫性に疑問
  • トランプ大統領自身が2025年夏の攻撃でイランの核計画は「壊滅した」と発言していた矛盾が指摘される
  • 米情報機関の評価ではイランのICBM開発は2035年まで必要との分析
  • ネタニヤフ首相が数カ月にわたりトランプ大統領に攻撃を促していたと複数メディアが報道
  • 共和党支持層内でも意見が分裂。攻撃が「イスラエルの策略」だったかを巡り論争
  • ランド・ポール上院議員（共和党）のみが戦争権限決議に賛成票。フェッターマン上院議員（民主党）のみが反対票

今後の焦点

注目すべきポイント

  • 後継指導者: イラン専門家会議が新最高指導者の選出を試みるも、イスラエルが「新指導者も標的」と警告。権力の空白が継続
  • 戦争の拡大: レバノン・ヒズボラの本格参戦、NATO加盟国（トルコ）への波及リスク
  • ホルムズ海峡: 米海軍によるタンカー護衛が実現するか。エネルギー危機の長期化リスク
  • 和平交渉: イラン側が和平交渉の接触を開始との報道あり。停戦の可能性
  • ロシア・中国: 両国がイラン支援に動くか、それとも距離を置き続けるか
  • 人道危機: 民間人被害の拡大、難民問題、インフラ破壊の影響
  • FIFAワールドカップ: イランサッカー連盟が2026年W杯（米・加・墨共催）参加の見通しは「良くない」と発言
